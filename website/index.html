Why Catzilla?

Built different. Benchmarked to prove it.

C-Powered Performance

HTTP engine written in C using libuv event loop and llhttp parser. Zero Python overhead in the hot path.

FastAPI-Compatible API

Drop-in familiar syntax. @app.get(), typed parameters, validation, dependency injection — all the patterns you know.

Memory Efficient

jemalloc arena specialization delivers 30–35% memory savings. Five dedicated arenas for requests, responses, cache, static files, and tasks.

Production Ready

Middleware, background tasks, file uploads, streaming, caching, and dependency injection. Everything you need, nothing you don't.

Familiar Syntax. Unfamiliar Speed.

If you know FastAPI, you already know Catzilla.

app.py 
from catzilla import Catzilla, JSONResponse, Request, Response

app = Catzilla()

@app.get("/")
def home(request: Request) -> Response:
    return JSONResponse({
        "message": "Hello from Catzilla 🐱",
        "speed": "72,249 req/s"
    })

if __name__ == "__main__":
    app.listen(port=8000)
